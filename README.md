## 📋 Descripción

Lexa Clothing es una tienda de moda online que ofrece ropa para hombre y mujer. El sitio cuenta con una página de inicio, secciones de colecciones por género, guía de talles y formulario de contacto con validación.

---

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura semántica con etiquetas `<main>`, `<section>`, `<article>`, `<nav>`
- **CSS3** — Flexbox, Grid, animaciones y media queries
- **SASS / SCSS** — Preprocesador CSS organizado en 6 parciales
- **JavaScript** — Validación de formulario y animaciones con ScrollReveal
- **Bootstrap 4** — Componentes y grilla responsive
- **Font Awesome** — Íconos

## 📄 Páginas

| Página | Descripción |
|---|---|
| `index.html` | Inicio — últimos drops y colección anterior |
| `hombre.html` | Sección masculina con galería de categorías |
| `mujer.html` | Sección femenina con grilla de productos |
| `talles.html` | Guía de talles con tablas interactivas |
| `contacto.html` | Formulario de registro con validación JS |

---

## ✨ Funcionalidades

- Diseño responsive para mobile, tablet y desktop
- Animaciones de entrada con ScrollReveal
- Navbar con submenú desplegable
- Galería con efecto hover en sección hombre
- Grilla de productos con CSS Grid en sección mujer
- Formulario con validación de campos en tiempo real
- Botón de scroll al inicio fijo

---

## 🚀 Cómo correr el proyecto

1. Cloná el repositorio
```bash
git clone https://github.com/Lexsa21/Lexa-Clothing.git
```
2. Abrí `index.html` con Live Server en VS Code o directamente en el navegador

> No requiere instalación de dependencias — es HTML, CSS y JS vanilla.
